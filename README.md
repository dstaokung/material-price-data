# ราคาวัสดุก่อสร้าง (Material Price Viewer)

เว็บแอปดูราคาวัสดุก่อสร้างรายเดือน/รายจังหวัด พร้อมส่งออกรายงาน PDF

**เปิดใช้งาน: https://dstaokung.github.io/material-price-data/** — v1.0

ข้อมูลอ้างอิงจาก [สำนักงานนโยบายและยุทธศาสตร์การค้า กระทรวงพาณิชย์ (tpso.go.th)](https://index.tpso.go.th)

## วิธีใช้

1. เลือก **เดือน** และ **จังหวัด** เพื่อโหลดข้อมูลราคา
2. กรองด้วยหมวดสินค้า หรือค้นหาชื่อวัสดุ
3. ติ๊กเลือกรายการที่ต้องการ แล้วกด **ส่งออก PDF**

## โครงสร้าง repo

| ตำแหน่ง | หน้าที่ |
|---|---|
| `index.html` | ตัวเว็บทั้งหมด (ไฟล์เดียว) — โหลดข้อมูลเฉพาะเดือน/จังหวัดที่เลือก |
| `data/<เดือนปี>/<จังหวัด>.json` | ข้อมูลราคาแยกตามเดือนและจังหวัด เช่น `data/062569/กทม.json` |

รูปแบบข้อมูลแต่ละแถว: `[รหัสหมวด, รหัสสินค้า, ชื่อรายการ, หน่วย, ราคา]`
